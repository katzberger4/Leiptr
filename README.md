# Leiptr
Measurements of chemical abundances in the Leiptr stellar stream taken during the Leadership Alliance SR-EIP 2023 at The University of Chicago

Leiptr Plots: magnesium abundance vs metallicity, log g vs T_eff, vt vs log g, [X/Fe] vs T_eff, [X/Fe] vs [Fe/H]
carretta_gc: Milky Way globular cluster data
iso_-1.5: isochrone for [Fe/H] = -1.5 using GAIA DR2 data at [a/Fe] = 0.4 and age = 12 Gyr
iso_-2.0: isochrone for [Fe/H] = -2.0 using GAIA DR2 data at [a/Fe] = 0.4 and age = 12 Gyr
iso_-2.49: isochrone for [Fe/H] = -2.49 using GAIA DR2 data at [a/Fe] = 0.4 and age = 12 Gyr
leipt-186_c_c_lesspayne: LESSPayne analysis for Leiptr-186 with temperature corrections and rebalanced stellar parameters
leipt-208_c_c_lesspayne: LESSPayne analysis for Leiptr-208 with temperature corrections and rebalanced stellar parameters
leipt-252_c_c_lesspayne: LESSPayne analysis for Leiptr-252 with temperature corrections and rebalanced stellar parameters
leipt-321_c_c_lesspayne: LESSPayne analysis for Leiptr-321 with temperature corrections and rebalanced stellar parameters
leipt-342_c_c_lesspayne: LESSPayne analysis for Leiptr-342 with temperature corrections and rebalanced stellar parameters
leiptr: Leiptr star names, GAIA IDs, radial velocities, stellar parameters, metallicities, and abundances
